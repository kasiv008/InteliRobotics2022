##README
#Code created by Kasi Viswanath for the Intelligent Robotics course work, Fall 2022.


Before running the code:
     1. Copy the turtlebot folder to the controller directory of your project.
     2. Load the final.wbt world file into the webots simulator.
     3. Define the DEF name for the robots as given in the controller (robot, rumba1, rumba2)
     4. Please set the Supervisor parameter true for all the robots in the world.
     5. Select turtlebot as the controller for the turtlebot3 robot.
     6. Select obstacle_avoidance controller for the create robot.
     7. Run the simulation.
     
If the goal is reached by the robot, a printed message 'Goal Reached' will appear.
